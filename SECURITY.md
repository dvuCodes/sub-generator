# Security Policy

## Supported versions

SubGen is currently source-first and under active development. Security fixes will be applied to the latest supported branch in this repository.

## Reporting a vulnerability

Please do not open a public GitHub issue for security-sensitive reports.

Use GitHub's private vulnerability reporting flow for this repository when it is enabled. If that flow is unavailable, contact the maintainer through the GitHub profile before any public disclosure and include:

- a clear description of the issue
- affected components or files
- reproduction steps or a proof of concept if available
- impact assessment
